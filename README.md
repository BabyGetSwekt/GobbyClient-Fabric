# GobbyClient-Fabric

A Hypixel SkyBlock Dungeons mod for Minecraft Fabric. Previously was a private Forge mod only for friends, now made for Fabric with new features and optimizations.
This mod is currently in development. DM @Goblinbanaan on Discord if you have any suggestions or bug reports (or make a PR/issue on GitHub)

## Features

### Dungeons

- **Brush** — Allows u to place blocks client-side in both boss and rooms (`/gobby brush`, `/gobby blockselector`)
- **Secret Triggerbot** — Autoclicks on secrets
- **Auto Close Chest** — Automatically closes chest screens in dungeons
- **Starred Mob ESP** - Highlights starred mobs & minibosses

### Floor 7

- **Lever Triggerbot** — Autoclicks levers in p3 (and has an option for lights device (s2), aswell)
- **Auto Align** — Aura's third device
- **Align Helper** — Prevents u from overclicking the frames
- **Auto Pre4** — Auto-aims and shoots emerald blocks on 4th device
- **Shooting Device ESP** — Highlights emerald blocks and completed shots on the 4th device
- **Debuff Area Renderer** — Visualizes debuff zones in p5
- **Last Breath Helper** — Automatically shoots last breath if you hold right click on dragons

It has more features, but these are the main ones. Use `/gobby` to open up the config.
